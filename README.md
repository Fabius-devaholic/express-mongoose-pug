# Express + Mongoose + Pug

This project uses express v4, mongoose v5, pug v2.

### Install
```
yarn
```

### Config
You should change config base on yours own on _.env-cmdrc_
```
vi .env-cmdrc
```

[pug engine](https://github.com/pugjs/pug),
